SimpleLink™ ZigBee® Wireless MCU

Z-Stack™ Home Software Development Kit Resource Guide

Version: 1.2.2a
October 16, 2015

The Texas Instruments Z-Stack™ Home 1.2.2a software development kit (SDK) includes all the necessary software to get started on the development of ZigBee® Home Automation applications using the SimpleLink portfolio of Wireless MCUs from Texas Instruments (CC2630/CC2650).

It includes object code with the ZigBee ® protocol stack, sample projects and applications, and several tools for the design and development of applications powered by ZigBee ® technology. Also included in this release is the TI-RTOS SimpleLink bundle and XDCTools.

In addition to the software, the kit contains the following documentation to aid in development

Notices

     - All CC2630 projects have been built and tested with IAR EWARM 7.40.2

     - The default TI-RTOS SimpleLink Bundle to use with this version of the SDK (and the version that is included in the installer) is: tirtos_simplelink_2_11_01_09.

     - The TI-RTOS SimpleLink Bundle default install path for this is: C:\ti\tirtos_simplelink_2_11_01_09.

     - The default XDCtools version to use with this version of the SDK (and the version that is included in the installer) is: xdctools_3_30_06_67_core.

     - The XDCtools default install path for this is: C:\ti\xdctools_3_30_06_67_core.

 

Z-Stack ™ SW has been certified as ZigBee Compliant Platform (ZCP) with binaries generated with the afore-mentioned toolchain and TI-RTOS SimpleLink version.
