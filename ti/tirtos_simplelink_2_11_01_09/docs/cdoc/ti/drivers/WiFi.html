WiFi Device type

Configuration settings
WiFi.deviceType = WiFi.DeviceType WiFi.DeviceType_SimpleLink_CC3000;
 
DETAILS
WiFi support is provided in the form of a libraries that are linked with your application. Several libraries are provided with the WiFi product to support the various devices. This configuration parameter allows you to select the proper library for the WiFi device to use.
The default value of deviceType is WiFi_DeviceType_SimpleLink_CC3000. For a complete list of options and what they offer see DeviceType.

RX payload size

Configuration settings
WiFi.rxPayloadSize = Int 1460;
 
DETAILS
Used to calculate the size of wlan_rx_buff[]--the buffer used to receive data and events from the Wi-Fi device. The maximum permitted size is 1460 bytes.Devices with limited memory may not be able to allocate the maximum payload size; smaller values can be used.
 
metaonly config WiFi.selectThreadPriority  // module-wide
index URL

SelectThread priority

Configuration settings
WiFi.selectThreadPriority = Int 1;
 
DETAILS
The SelectThread is a task created by the multi-thread version of the host driver to prevent long host driver calls such as recv() and accept() from preventing other tasks from making host driver API calls while they wait for a response.
You may select the Task priority as you would for any other Task instance.
